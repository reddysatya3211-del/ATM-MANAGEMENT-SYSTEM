# ATM-MANAGEMENT-SYSTEM
This is a program that shows how a real ATM works.This program was written in PYTHON. It duplicates the real time ATM functions such as: Ballannce enquiry, Deposits money,Withdraw money,Showing account details,etc...The main purpose of this project is to show how banking transactions can be performed digtally.And creating a secure PIN based access to a bank acccount.and to learn how OOPs concepts work in python.
This is ideal for:
*School or College mini projects
*Python OOP learning
How it runs:
When the program runs,it welcomes the user and asks for a PIN.
If the pin is correct,then the ATM becomes active.
Then a menu appears with multiple options,then the user need to select an option from the given options
*View balance
*Deposit cash
*Withdraw cash
*Check account details
*Exit
The loop will continue untill the user chooses the exit option.
If the entered pin was incorrect,then the ACCESS DENIED option will be displayed.
STRUCTURE OF THE PROGRAM
This program uses 3 classes.They are 1.Abstract class 2.Parent class 3.Child class.
And these are responsible for
1. Validating amounts
2. Showing bank name
3. Deposits
4. Withdrawls
5. ATM withdrawl limit
6. Displaying balance.
How to run the program
1. Enter pin
2. Choose actions through menu.
